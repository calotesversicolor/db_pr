## 1

```sql
SELECT menu.id AS object_id, pizza_name AS object_name FROM menu
UNION
SELECT person.id, name FROM person
ORDER BY object_id ASC, object_name ASC;
```

## 2

```sql
SELECT menu.id AS object_id, pizza_name AS object_name FROM menu
UNION All
SELECT person.id, name FROM person
ORDER BY object_name DESC;
```
